# Mini Shopify

Mini Shopify is a full-stack SaaS e-commerce platform inspired by Shopify.

Merchants can create stores, manage products, receive orders, and view weekly analytics. Customers can browse public storefronts, add products to cart, and place orders.

## Tech Stack

- Backend: Java 21, Spring Boot 3
- Frontend: React, TypeScript, Vite
- Database: PostgreSQL
- Cache: Redis later
- Storage: MinIO/S3 later
- Deployment: Docker Compose

## Core Features

- Merchant authentication
- Store creation
- Product management
- Public storefront
- Cart and checkout
- Order management
- Weekly analytics dashboard
